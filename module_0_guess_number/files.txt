README.md - файл с описанием проекта, целей и результатов

requirements.txt - файл с версиями библиотек на момент создания проекта. Получен с помощью pip freeze.

game.py - файл с самой первой, простейшей версией "угадайки"

game_v2.py - файл с двумя более продвинутыми версиями "угадайки": случайным угадыванием с учётом "больше-меньше" и бинарным поиском.

game.ipynb - файл jupiter notebook, содержащий только результаты угадывания двух алгоритмов
